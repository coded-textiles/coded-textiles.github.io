Description

Rows 0 and 1 represent generation 0. Rows 2 and 3 represent generation 1, and so on. The first generation is randomly generated. Every generation after that includes two children, each produced by combining the parent generations:

g0a + g0b -> g1a
g0b + g0a -> g1b
g1a + g1b -> g2a
g1b + g1a -> g2b...

In this example, mutations are not inherited.
