Description

Each row is a generation, with a population of 2. The first generation is randomly generated. Every generation after that includes two children, each produced by combining the parent generations:

r1a + r1b -> r2a
r1b + r1a -> r2b

r2a + r2b -> r3a
r2b + r2a -> r3b...


All horizontal and vertical translation "mutations" are inherited.
