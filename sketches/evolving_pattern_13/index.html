Description

Each row pair (e.g. row 1 & 2, 3 & 4, etc.) is a generation, with a population of 2. The first generation is randomly generated. Every generation after that includes two children, each produced by combining the parent generations:

r1 + r2 -> r3
r2 + r1 -> r4

r3 + r4 -> r5
r4 + r3 -> r6...

All horizontal and vertical translation "mutations" are inherited.
